# Logo Recognition System

Система детекции и распознавания логотипов на изображениях с использованием глубокого обучения.

## О проекте
Данный проект реализует автоматическое обнаружение логотипов и их классификацию по брендам на изображениях одежды.

Решение построено на двухэтапной архитектуре:
   1) Детекция логотипов с помощью нейросети YOLOv8.
   2) Классификация бренда каждого найденного логотипа с использованием сверточной нейросети EfficientNet.

Проект разработан в рамках учебной работы и демонстрирует полный цикл построения системы компьютерного зрения: от подготовки датасета до инференса модели.

Ключевая особенность:

Использование каскадного подхода (Detection → Classification), который позволяет:

   • находить несколько логотипов на одном изображении,

   • классифицировать каждый логотип независимо,

   • легко масштабировать систему под новые бренды.

## Функциональность
### 1. Предобработка данных

• Загрузка датасета LogoDet-3K (Clothes).

• Конвертация аннотаций из XML (Pascal VOC) в формат YOLO.

• Автоматическое разбиение на train_toggle (80%) и validation (20%).

### 2. Детекция логотипов

• Модель: YOLOv8n (Ultralytics)

• Задача: локализация логотипов на изображении.

• Аугментации:

   1) Mosaic

   2) MixUp

   3) Случайные преобразования масштаба

### 3. Генерация кропов

• Все найденные логотипы автоматически вырезаются (cropping).

• Кропы группируются по брендам.

• Полученный датасет используется для обучения классификатора.

### 4. Классификация брендов

• Модель: EfficientNet-B0 (pretrained)

• Вход: кроп логотипа

• Выход: класс бренда

• Оптимизатор: Adam

• Loss: CrossEntropy

### 5. Инференс (распознавание)

Для каждого изображения:

   1) YOLO находит логотипы.

   2) Каждый логотип классифицируется.

   3) На выходе — изображение с bounding box + названием бренда и вероятностью.

## Датасет
Проект использует открытый датасет:

LogoDet-3K (Clothes subset)
Содержит изображения одежды с разметкой логотипов различных брендов.

## Пример результата
На выходе система автоматически определяет логотип и бренд:

   • Bounding box вокруг логотипа

   • Название бренда

   • Уверенность модели (probability)
